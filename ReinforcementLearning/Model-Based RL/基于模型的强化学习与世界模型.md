# 基于模型的强化学习与世界模型

## 概述

基于模型的强化学习（Model-Based Reinforcement Learning, MBRL）是强化学习中一类核心方法，其核心思想是：**先学习环境的动态模型，再利用模型进行规划或策略优化**。相比 Model-Free 方法（如 [[TD]] 中的 Q-learning、[[PolicyBased]] 中的 PPO），MBRL 通常具有更高的**样本效率**（sample efficiency），因为智能体可以在"想象"中反复练习，而不必每次都与真实环境交互。

前置知识参见 [[1.introduction（RL）]]、[[Dyna-Q]]。

> [!abstract] 本文结构
> 1. Model-Free vs Model-Based 对比
> 2. 经典 MBRL 方法（Dyna → PILCO → PETS → MBPO）
> 3. 世界模型方法（World Models → MuZero → Dreamer 系列 → IRIS → TD-MPC → DIAMOND）
> 4. 基础世界模型与前沿进展（UniSim、Genie、Cosmos 等）
> 5. 核心技术总结与方法对比

## 1. Model-Free vs Model-Based

### 核心区别

| 维度 | Model-Free | Model-Based |
|------|-----------|------------|
| **是否学习环境模型** | ❌ 不学习 | ✅ 学习 $\hat{p}(s', r \mid s, a)$ |
| **数据来源** | 仅真实交互 | 真实交互 + 模型模拟 |
| **样本效率** | 低（需大量交互） | 高（模型可生成无限模拟经验） |
| **渐近性能** | 通常更高（无模型偏差） | 受限于模型精度 |
| **计算开销** | 低（无需训练模型） | 高（需训练模型 + 规划） |
| **核心挑战** | 探索效率 | 模型误差累积（compounding error） |

### MBRL 的统一框架

所有 MBRL 方法都遵循一个共同范式：

```
┌───────────────────────────────────────────────────────┐
│              Model-Based RL 统一框架                    │
│                                                       │
│  1. 收集真实经验    D_real = {(s, a, r, s')}          │
│  2. 学习环境模型    M_θ ≈ p(s', r | s, a)             │
│  3. 利用模型规划    生成模拟经验 / 直接优化策略          │
│  4. 更新策略        π ← improve(π, D_real ∪ D_model)  │
│  5. 重复 1-4                                          │
└───────────────────────────────────────────────────────┘
```

方法之间的差异主要体现在：
- **模型类型**：确定性 / 概率性 / 集成 / 潜空间 / 扩散模型
- **规划方式**：随机射击 / MPC / 潜空间想象 / 树搜索
- **策略学习**：在模型内训练 Actor-Critic / 直接搜索 / 反向传播

## 2. 经典 MBRL 方法

### 2.1 Dyna（1991）

> 详细笔记见 [[Dyna-Q]]

Dyna 是 Sutton 提出的最早的 MBRL 框架，其核心思想极为优雅：**Direct RL 和 Planning 使用完全相同的更新公式**，唯一区别是数据来源。

$$
Q(s, a) \leftarrow Q(s, a) + \alpha \left[ r + \gamma \max_{a'} Q(s', a') - Q(s, a) \right]
$$

- 真实经验 → Direct RL
- 模型生成的经验 → Planning

**局限性**：模型是查找表（tabular），确定性假设，无法处理高维连续空间。

---

### 2.2 PILCO（2011）

**PILCO**（Probabilistic Inference for Learning Control）是首个将**高斯过程（GP）**引入 MBRL 的代表性工作，由 Deisenroth & Rasmussen 提出。

**核心思想**：

1. **概率动态模型**：使用高斯过程学习状态转移函数 $f \sim \mathcal{GP}$，天然提供**不确定性估计**
2. **解析策略梯度**：通过解析传播不确定性来计算长期累积奖励的梯度
3. **直接优化策略**：通过梯度下降直接优化参数化策略 $\pi_\phi$

$$
J(\phi) = \sum_{t=0}^{T} \mathbb{E}\left[ c(s_t) \right], \quad s_{t+1} \sim \mathcal{GP}(f | s_t, \pi_\phi(s_t))
$$

**关键优势**：
- **极致的样本效率**：在 cart-pole 等任务上只需几十条真实轨迹即可学习
- **不确定性感知**：模型不确定性直接影响策略更新，避免在不确定区域过度利用

**局限性**：
- GP 的计算复杂度为 $O(n^3)$，无法扩展到高维状态/动作空间
- 不适用于图像观测等高维输入

> [!quote] 参考文献
> Deisenroth, M. P. & Rasmussen, C. E. (2011). PILCO: A Model-Based and Data-Efficient Approach to Policy Search. *ICML 2011*.

---

### 2.3 PETS（2018）

**PETS**（Probabilistic Ensemble Trajectory Sampling）解决了 PILCO 的可扩展性问题。

**核心改进**：

1. **集成概率神经网络**：用多个概率神经网络（ensemble of probabilistic NNs）替代 GP
   $$
   f_{\theta_i}(s, a) = (\mu_i, \sigma_i^2), \quad i = 1, \ldots, B
   $$
2. **TS1 采样（Trajectory Sampling）**：在 rollout 时，每一步随机选择集成中的一个模型，通过 bootstrap 采样传播**认知不确定性（epistemic uncertainty）**
3. **CEM + MPC 规划**：使用交叉熵方法（Cross-Entropy Method）在每一步搜索最优动作序列

```
PETS 规划流程：
  For each time step:
    1. 用 CEM 采样 N 个候选动作序列
    2. 对每个序列，用集成模型 rollout 预测轨迹
    3. 评估每条轨迹的累积奖励
    4. 用 top-k 序列更新 CEM 分布
    5. 执行最优序列的第一个动作（MPC）
```

**关键突破**：首次在高维连续控制任务上展示了 MBRL 可以匹配甚至超越 Model-Free 方法的样本效率。

> [!quote] 参考文献
> Chua, K. et al. (2018). Deep Reinforcement Learning in a Handful of Trials using Probabilistic Dynamics Models. *NeurIPS 2018*.

---

### 2.4 MBPO（2019）

**MBPO**（Model-Based Policy Optimization）是将 Model-Based 和 Model-Free 优雅结合的里程碑之作，由 UC Berkeley 的 Janner 等人提出。

**核心思想**：使用学习到的模型生成**短 rollout** 数据，然后用 Model-Free 算法（SAC）在混合数据上训练。

**关键设计**：

1. **短分支 rollout（Branched Rollouts）**：
   - 从真实经验池中采样真实状态 $s_{\text{real}}$
   - 从 $s_{\text{real}}$ 出发，用模型 rollout $k$ 步（通常 $k = 1 \sim 5$）
   - 短 rollout 有效控制了模型误差累积

2. **集成概率模型**：与 PETS 类似，使用概率集成模型

3. **理论保证**：
   提供了单调改进的理论下界。rollout 长度 $k$ 的选择取决于模型误差 $\epsilon_m$ 和策略偏移：
   $$
   \eta[\pi] \geq \hat{\eta}[\pi] - C \cdot \left( \epsilon_m + \epsilon_\pi \right)
   $$
   其中 $\hat{\eta}$ 是模型上的策略性能，$C$ 与 rollout 长度相关。

4. **混合数据训练**：
   $$
   D_{\text{train}} = D_{\text{real}} \cup D_{\text{model}}
   $$
   在混合数据上运行 SAC，兼获 Model-Based 的样本效率和 Model-Free 的渐近性能。

**性能表现**：
- 在 MuJoCo 连续控制任务上，以 Model-Free 方法 **1/10 的数据量** 达到相同的渐近性能
- 扩展到了以前 MBRL 方法无法处理的高维状态空间

> [!quote] 参考文献
> Janner, M. et al. (2019). When to Trust Your Model: Model-Based Policy Optimization. *NeurIPS 2019*.
>
> BAIR Blog: [Model-Based Reinforcement Learning: Theory and Practice](https://bair.berkeley.edu/blog/2019/12/12/mbpo/)

## 3. 基于世界模型的强化学习

"世界模型"（World Model）是 MBRL 的进阶形态——不再只是学习简单的状态转移函数，而是学习一个**压缩的、可模拟的世界表征**，使智能体能够在**潜空间（latent space）**中"想象"未来并据此行动。

> [!tip] 核心哲学
> "Learn a compact world. Then learn to act inside that world. Only after mastering the dream should you step into reality."

---

### 3.1 World Models（2018）— 开山之作

Ha & Schmidhuber 的 **World Models** 论文正式将"世界模型"概念引入深度 RL，奠定了整个领域的范式。

**三组件架构**：

```
┌──────────────────────────────────────────────────────┐
│                  World Models 架构                    │
│                                                      │
│  观测 o_t ──→ [V: VAE 编码器] ──→ z_t（潜表征）      │
│                                       │              │
│                                       ▼              │
│               [M: MDN-RNN] ──→ 预测 z_{t+1}          │
│                    │                                  │
│                    ▼                                  │
│               [C: Controller] ──→ 动作 a_t            │
│                                                      │
└──────────────────────────────────────────────────────┘
```

| 组件 | 模型 | 功能 |
|------|------|------|
| **V**（Vision） | VAE | 将高维观测压缩为低维潜向量 $z_t$ |
| **M**（Memory） | MDN-RNN | 预测潜空间中的下一状态分布 $p(z_{t+1} \mid z_t, a_t)$ |
| **C**（Controller） | 线性控制器 | 根据 $z_t$ 和 $h_t$（RNN 隐状态）输出动作 |

**训练流程**：
1. 随机策略收集数据 → 训练 V（VAE）
2. 在潜空间训练 M（MDN-RNN）
3. 在 M 的"梦境"中用进化策略训练 C

**意义**：首次证明**完全在梦中训练的策略可以迁移到真实环境**（在 VizDoom、赛车游戏中验证）。

> [!quote] 参考文献
> Ha, D. & Schmidhuber, J. (2018). World Models. *arXiv:1803.10122*.

---

### 3.2 MuZero（2020）— 规划与学习的统一

**MuZero**（Mastering Atari, Go, Chess and Shogi by Planning with a Learned Model）由 DeepMind 提出，是 AlphaGo → AlphaZero 路线的终极进化。

**核心突破**：不需要环境规则，学习一个**隐式环境模型**用于蒙特卡洛树搜索（MCTS）。

**三个网络**：

$$
\text{Representation: } h_\theta: o_t \to s^0
$$
$$
\text{Dynamics: } g_\theta: (s^k, a^{k+1}) \to (r^{k+1}, s^{k+1})
$$
$$
\text{Prediction: } f_\theta: s^k \to (p^k, v^k)
$$

```
观测 o_t ──→ [Representation h] ──→ s^0 ──→ [Prediction f] ──→ (policy p^0, value v^0)
                                      │
                              a^1     ▼
                            ──→ [Dynamics g] ──→ (r^1, s^1) ──→ [Prediction f] ──→ (p^1, v^1)
                                                    │
                                            a^2     ▼
                                          ──→ [Dynamics g] ──→ (r^2, s^2) ──→ ...
```

**关键设计**：
- **端到端训练**：三个网络联合训练，损失函数结合策略、价值和奖励预测
- **MCTS 规划**：在学习到的潜空间中执行树搜索，无需真实模拟器
- **不要求状态重建**：Dynamics 网络只需产生对**规划有用**的表征，不需要重建原始观测

**性能**：
- 在围棋、国际象棋、将棋上**匹配 AlphaZero**（后者需要已知规则）
- 在 57 个 Atari 游戏上**刷新 SOTA**
- 比 AlphaZero 减少 20% 搜索计算量

> [!quote] 参考文献
> Schrittwieser, J. et al. (2020). Mastering Atari, Go, Chess and Shogi by Planning with a Learned Model. *Nature*, 588, 604–609.

---

### 3.3 Dreamer 系列（2020–2025）— 潜空间想象的巅峰

Dreamer 系列是目前**最成功的通用世界模型 RL 框架**，由 Hafner 等人持续迭代。

#### DreamerV1（2020）

首次实现了**完全在潜空间想象中训练 Actor-Critic** 的方案。

**RSSM（Recurrent State-Space Model）**：

$$
\text{序列模型: } h_t = f_\phi(h_{t-1}, z_{t-1}, a_{t-1})
$$
$$
\text{编码器: } z_t \sim q_\phi(z_t \mid h_t, o_t) \quad \text{（后验）}
$$
$$
\text{动态先验: } \hat{z}_t \sim p_\phi(\hat{z}_t \mid h_t) \quad \text{（先验）}
$$
$$
\text{解码器: } \hat{o}_t \sim p_\phi(\hat{o}_t \mid h_t, z_t)
$$
$$
\text{奖励预测: } \hat{r}_t \sim p_\phi(\hat{r}_t \mid h_t, z_t)
$$

**想象训练**：
1. 从真实经验训练 RSSM 世界模型
2. 从真实状态出发，在模型中展开想象轨迹（imagination rollout）
3. 在想象轨迹上训练 Actor（策略）和 Critic（价值函数）
4. 通过重参数化梯度反向传播优化策略

> [!quote] 参考文献
> Hafner, D. et al. (2020). Dream to Control: Learning Behaviors by Latent Imagination. *ICLR 2020*.

#### DreamerV2（2021）

关键改进：将潜变量从连续高斯分布改为**离散类别分布**。

$$
z_t \in \{1, \ldots, K\}^D \quad \text{（D 个独立的 K 类类别变量）}
$$

- **为什么用离散**：离散表征更适合捕捉多模态环境动态，且梯度通过 straight-through estimator 传播
- **性能**：首个在 Atari 55 游戏上超越人类平均水平的世界模型方法

> [!quote] 参考文献
> Hafner, D. et al. (2021). Mastering Atari with Discrete World Models. *ICLR 2021*.

#### DreamerV3（2023）

**通用性突破**：一套超参数适用于 150+ 不同任务域。

**关键技术**（鲁棒性三件套）：

| 技术 | 说明 |
|------|------|
| **Symlog 预测** | 对称对数变换处理不同量级的奖励和价值 |
| **百分位回报归一化** | 将回报归一化到 $[0, 1]$，消除不同任务间的量纲差异 |
| **Free bits** | KL 散度下界，防止先验坍缩 |

**标志性成就**：
- **Minecraft 钻石挑战**：首个从零开始在 Minecraft 中获得钻石的 RL 算法
- **跨域通用**：从 Atari 像素游戏到 DMLab 3D 导航到机器人控制，无需调参
- 发表于 *Nature*（2025 年正式刊发）

> [!quote] 参考文献
> Hafner, D. et al. (2023). Mastering Diverse Domains through World Models. *arXiv:2301.04104*; *Nature* (2025).

### 3.4 IRIS（2023）— 基于 Transformer 的离散世界模型

**IRIS**（Imagination with auto-Regression over an Inner Speech）用 **Autoregressive Transformer** 替代 RSSM 来建模世界。

**架构**：

1. **离散化**：用 dVAE（discrete VAE）将每帧图像编码为离散 token 序列
2. **自回归世界模型**：GPT 风格的 Transformer 自回归预测下一帧的 token
3. **在 Transformer "梦境"中训练策略**

$$
o_t \xrightarrow{\text{dVAE}} (z_t^1, z_t^2, \ldots, z_t^K) \xrightarrow{\text{Transformer}} (\hat{z}_{t+1}^1, \hat{z}_{t+1}^2, \ldots, \hat{z}_{t+1}^K)
$$

**特点**：
- 利用 Transformer 的强大序列建模能力
- 在 Atari 100k benchmark 上取得有竞争力的结果
- 但离散 token 可能丢失重要视觉细节

> [!quote] 参考文献
> Micheli, V. et al. (2023). Transformers are Sample-Efficient World Learners. *ICLR 2023*.

---

### 3.5 TD-MPC / TD-MPC2（2022/2024）— 潜空间中的模型预测控制

**TD-MPC**（Temporal Difference Model Predictive Control）将经典 MPC 与深度潜空间模型结合。

**核心思想**：联合学习潜状态表征、确定性动态模型、奖励函数和终值函数，然后用 MPC 在潜空间中在线规划。

**架构组件**：

$$
\text{编码器: } z_t = h_\theta(o_t)
$$
$$
\text{动态模型: } z_{t+1} = d_\theta(z_t, a_t) \quad \text{（确定性）}
$$
$$
\text{奖励预测: } \hat{r}_t = R_\theta(z_t, a_t)
$$
$$
\text{终值函数: } V_\theta(z_t) \quad \text{（通过 TD 学习）}
$$
$$
\text{策略先验: } a_t = \pi_\theta(z_t) \quad \text{（引导 MPC 搜索）}
$$

**规划过程**（在线 MPC）：
1. 用 CEM/MPPI 在潜空间中搜索最优动作序列
2. 目标函数 = 短期奖励累积 + 终值函数估计
3. 策略网络提供搜索初始化（warm start），大幅加速规划
4. 执行第一个动作，下一步重新规划

**TD-MPC2 改进**：
- 确定性动态模型（更稳定）
- 借鉴 DreamerV3 的归一化技术
- 单一模型架构覆盖 80+ 不同任务
- 在连续控制任务上达到 SOTA

> [!quote] 参考文献
> Hansen, N. et al. (2022). Temporal Difference Learning for Model Predictive Control. *ICML 2022*.
> Hansen, N. et al. (2024). TD-MPC2: Scalable, Robust World Models for Continuous Control. *ICLR 2024*.

---

### 3.6 DIAMOND（2024）— 扩散世界模型

**DIAMOND**（DIffusion As a Model Of eNvironment Dreams）首次将**扩散模型**（Diffusion Model）引入世界模型，在 NeurIPS 2024 获得 Spotlight。

**动机**：此前的离散 token 世界模型（如 IRIS）在量化过程中会丢失视觉细节，而这些细节对决策至关重要（如 Atari 中小球的精确位置）。

**核心方法**：

1. **扩散动态模型**：世界模型是一个条件扩散模型，给定历史帧和动作，去噪生成下一帧
   $$
   p_\theta(o_{t+1} \mid o_{t-H:t}, a_{t-H:t}) = \text{DiffusionProcess}(\theta)
   $$
2. **在扩散"梦境"中训练**：智能体完全在扩散模型生成的想象轨迹中学习策略

**关键设计**：
- 使用 **EDM**（Elucidated Diffusion Models）框架
- 去噪步数优化以平衡质量和速度
- 仔细的噪声调度避免错误累积

**性能**：
- Atari 100k 基准：**平均人类归一化分数 1.46**（超越 IRIS 等方法）
- 生成的画面**视觉质量显著优于** token-based 方法
- 证明了"更好的视觉建模 → 更好的决策"

> [!quote] 参考文献
> Alonso, E. et al. (2024). Diffusion for World Modeling: Visual Details Matter in Atari. *NeurIPS 2024 Spotlight*.

## 4. 基础世界模型（Foundation World Models）— 前沿方向

近年来，大规模预训练世界模型正在成为新的研究前沿，这些模型的目标是构建**通用的环境模拟器**，而不仅限于特定 RL 任务。

### 4.1 UniSim（2023）

Google 的 **UniSim** 是一个**通用世界模拟器**，能够根据多种输入条件动态生成视频序列。

- **多模态条件**：空间运动、文本指令、相机参数
- **多源数据训练**：3D 仿真数据 + 真实机器人数据 + 互联网视频
- **应用**：为机器人操控生成训练数据，实现 sim-to-real 迁移

### 4.2 Genie 系列（2024–2025）

Google DeepMind 的 **Genie** 系列是**交互式世界生成模型**：

- **Genie 1**（2024）：110 亿参数，从单张图像生成可玩的 2D 平台游戏环境
- **Genie 2/3**（2025）：支持文本到世界的实时交互生成，可用于生成 RL 训练课程

**架构核心**：
- 视频 Tokenizer + 潜动作模型 + 动态模型
- 学习无标注的"潜在动作空间"，使模型具备可控交互性

### 4.3 NVIDIA Cosmos（2025）

**Cosmos** 是 NVIDIA 发布的**开放权重世界基础模型平台**：

- 专注于 **3D 一致性**和**物理对齐**
- 面向自动驾驶和机器人仿真
- 提供大规模预训练的视频生成模型

### 4.4 V-JEPA 2（2025）

Meta 的 **V-JEPA 2** 展示了**网络规模视频预训练**可以产生可操作的世界模型：

- 在大规模视频数据上预训练联合嵌入预测架构
- 少量机器人数据微调即可用于机器人控制
- 同时支持视频理解和动作预测

> [!info] 趋势
> 基础世界模型正在将"世界理解"与"视频生成"两大能力融合，未来 RL 智能体可能直接在这些通用世界模型中训练，大幅降低对特定环境的依赖。

## 5. 核心技术总结

### 5.1 环境模型的类型

| 模型类型 | 代表方法 | 优势 | 劣势 |
|---------|---------|------|------|
| **确定性模型** | Dyna, TD-MPC | 简单高效 | 无法表征不确定性 |
| **高斯过程** | PILCO | 天然不确定性估计 | $O(n^3)$ 不可扩展 |
| **概率集成** | PETS, MBPO | 可扩展的不确定性 | 集成训练开销大 |
| **RSSM（潜变量）** | Dreamer 系列 | 压缩高维观测 | 训练复杂 |
| **自回归 Transformer** | IRIS | 强大的序列建模 | 离散化损失细节 |
| **扩散模型** | DIAMOND | 高保真视觉生成 | 推理速度慢 |
| **隐式模型（不重建）** | MuZero | 只学对规划有用的 | 可解释性差 |

### 5.2 规划方式

```
规划方式分类：

├── 背景规划（Background Planning）
│   ├── Dyna：随机采样回放 + Q-learning 更新
│   └── Dreamer：潜空间 rollout + Actor-Critic 训练
│
├── 决策时规划（Decision-Time Planning）
│   ├── MPC 类：PETS、TD-MPC（在线优化动作序列）
│   └── 树搜索：MuZero（MCTS + 学习模型）
│
└── 混合：MBPO（模型生成数据 + SAC 离线训练）
```

### 5.3 处理模型误差的策略

| 策略 | 方法 | 说明 |
|------|------|------|
| **短 rollout** | MBPO | 只用模型展开 1-5 步，限制误差累积 |
| **集成不确定性** | PETS, MBPO | 多模型投票，不确定时保守行动 |
| **潜空间建模** | Dreamer, TD-MPC | 在压缩空间建模，降低预测难度 |
| **KL 正则化** | Dreamer | 约束先验和后验一致性 |
| **扩散去噪** | DIAMOND | 通过迭代去噪保留细节 |
| **不重建观测** | MuZero | 只优化对规划有用的信息 |

## 7. 发展脉络与未来展望

```
时间线：

1991  Dyna ──────────── 模型辅助规划的开创
        │
2011  PILCO ─────────── 概率模型 + 解析梯度
        │
2018  PETS ──────────── 集成 NN + MPC
      World Models ──── VAE-RNN-Controller 范式
        │
2019  MBPO ──────────── 短 rollout + SAC 混合
        │
2020  MuZero ────────── 隐式模型 + 树搜索
      DreamerV1 ─────── RSSM + 潜空间 AC
        │
2021  DreamerV2 ─────── 离散潜变量
        │
2023  DreamerV3 ─────── 通用跨域
      IRIS ──────────── Transformer 世界模型
        │
2024  TD-MPC2 ────────── 通用潜空间 MPC
      DIAMOND ────────── 扩散世界模型
        │
2025  Genie 3, Cosmos ── 基础世界模型
      V-JEPA 2 ────────── 视频预训练世界模型
        │
 未来  通用世界模拟器 ── RL 智能体在通用世界模型中训练？
```

> [!important] 未来关键问题
> 1. **模型可扩展性**：能否训练一个"世界的 GPT"——在海量视频上预训练、适用于任意下游 RL 任务？
> 2. **长期一致性**：现有世界模型在长 rollout 时仍会退化，如何保持时序一致性？
> 3. **物理对齐**：如何让世界模型遵守物理定律，而非仅拟合表面统计？
> 4. **计算效率**：扩散模型和大规模 Transformer 的推理成本仍是瓶颈
> 5. **迁移与泛化**：从预训练世界模型到下游任务的高效迁移
